# Sistema de Corte Automático de Vídeos

Projeto desenvolvido em Python utilizando MoviePy para automatizar a preparação de vídeos para cortes.

O sistema ajusta automaticamente as dimensões do vídeo, adiciona elementos visuais e gera um novo vídeo pronto para publicação.

## Funcionalidades

- Leitura de vídeos MP4
- Ajuste automático de resolução
- Redimensionamento proporcional
- Corte automático (crop)
- Inserção de textos
- Inserção de imagem de fundo
- Composição de vídeo em camadas
- Exportação para MP4

## Tecnologias Utilizadas

- Python
- MoviePy
- ImageMagick
- Google Colab
